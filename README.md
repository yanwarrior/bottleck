# bottleck
Bottleck Custom To MVC Pattern

##### How To Use ?
```
$ pip install -r requirements.txt
```

##### How To Running ?
```
$ python run.py

Bottle v0.12.8 server starting up (using WSGIRefServer())...
Listening on http://0.0.0.0:8080/
Hit Ctrl-C to quit.
```
